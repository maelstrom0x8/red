# red
### OAuth2 Client with Spring Boot

## Prerequisites
- Java 21

## Quick Start

```bash
./mvnw spring-boot:test-run
```

This application use GitHub as it's default authorization provider.

#### Environment Variables
- GH_CLIENT_ID
- GH_CLIENT_SECRET
- PORT  // defaults to 8080
